Bem-vindo ao JS Quiz

Seu repositório de testes em JavaScript

Primeiramente, vamos começar com um teste simples. Precisamos que você retorne a palavra QUIZ.

Para fazer isso, basta retornar a string contendo a palavra QUIZ no CodeFrame e clique em Testar, para que o aplicativo teste seu código. Em seguida, se tudo der certo, você deve clicar em Submeter. Assim o código que você submeteu, será testado e finalizado, dando a oportunidade de pontuar e continuar a fazer os demais testes.

Exemplo: return 'QUIZ';
